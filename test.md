# 1. Python설치하기

1) Windows 에서 키보드의 windows 키를 누르고 powershell을 입력하고, 앱이 하나 검색되는데 실행시켜봅니다.

2) 아래와같은 창이 하나 뜰텐데요, 여기에 python --version 이라고 입력해 보겠습니다.
&ensp;![](./.images/221129_Jupyterlab_install_1.png)  

1) Python이 설치되어있지 않으면 위와같이 뜨는게 정상입니다. Python을 설치할게요.

2) [Python Releases for Windows | Python.org](https://www.python.org/downloads/windows/)  
   위 사이트에 접속하셔서 'Latest Python3 Release'를 선택하시고 내려가시면 windows 64bit용(32bit 사용사이시면 32bit 선택) 다운로드가 있습니다.
    (Windows App store에서도 설치가 가능하지만, 위 방법을 추천합니다. store에서 설치하는 경우 환경변수 설정 등 추가 작업이 필요합니다.)

5) 다운받은 exe를 실행시키세요. 

6) 실행시, 아래 Add python.exe to PATH에도 체크하시고, Install Now를 눌러서 설치를 진행하시면 됩니다.
&ensp;![](./.images/221129_Jupyterlab_install_2.png)  

7) Powershell로 돌아가서 python --version을 다시 입력해보세요. (Powershell도 재실행 하셔야됩니다.)

&nbsp;와. 벌써 Python 설치가 끝났네요. 아까 설치시에 다운로드 받았던 최신버전의 Python이 설치됐다고 나옵니다.

# 2. Jupyter lab 설치하기

Python은 설치가 끝났습니다. 이제 Jupyter lab을 설치할 차례입니다. python은 pip라는 도구를 통해서 필요한 package를 설치합니다. 

위에 보시면 현재 제 pc엔 2가지 밖에 없으니, 필요한 jupyter lab을 설치해야합니다.

1) powershell로 돌아가서, pip install jupyterlab 이라고 입력하시면 (jupyterlab 띄어쓰기없음!!)
&ensp;![](./.images/221129_Jupyterlab_install_4.png)  

&ensp;&ensp;올게 왔네요. 'Connection 어쩌고 저쩌고,,', '원격 호스트에 어쩌고 저쩌고..' 이런거 나오면 보통 '아..proxy 구나' 하시면 됩니다. proxy를 해결해 보겠습니다.

2) 준비물이 필요합니다. 각 사업장별 '인증서(crt)'파일입니다. 보통 ESCORT 트레이아이콘 마우스 우클리 -> PC-Download에 가셔서 

&ensp;&ensp;`[ 03. 필수설치 인터넷설정 / 04. 프록시(Proxy) ]` 쪽에 보시면 '수동설치'가 있습니다. 누르면 crt 파일이 다운로드 됩니다. (제 PC의 다운로드 폴더에 저장할게요)

3) pip install을 다시 시도합니다. 명령어는 이제 pip install jupyterlab 이 아닌 

&ensp;&ensp;`pip install --cert <crt file path> --proxy <proxy주소> jupyterlab` 이렇게 좀 길게 바뀝니다.

**1. 시험환경과 동일한 python 3.7.4 버전을 설치합니다.**   

&ensp;1-1. [Python Release Python 3.7.4 | Python.org](https://www.python.org/downloads/release/python-374/) 에서 다운받아 Customize installation 을 통해 설치해 줍니다. (본인 PC 에 이미 다른 버전의 python 이 install 되어 있더라도 관계없이 설치해 줍니다.)

&ensp;1-2. python 설치 경로를 C:\python37로 합니다. (굳이 설치 경로를 따를 필요는 없지만, 설명은 C:\python37 를 기준으로 합니다. 다른 옵션은 체크하지 않고 default 옵셥만 체크한체로 install 해줍니다.)

**2. 만일 이전에 한 번도 python 을 설치한 적이 없었다면 2번을 통해 python 설치 후 3번의 가상환경 설정 필요없이 곧바로 4-2의 pip install -r requirements.txt 를 실행해주시면 되겠습니다.**   
$\textsf{\color{#FA3D7E}{(이전에 다른 버전의 python 설치해서 사용하셨고, 계속 사용하셔야 하시면 3번으로 가시면 되겠습니다.)}}$
